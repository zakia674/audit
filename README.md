# 🛡️ Learn=>Audit : ISO 2700x

## 📌 Description

Learn=>Audit est une application web qui simule un audit de sécurité des systèmes d'information basé sur les normes ISO 2700x.
Elle permet à un auditeur de gérer les missions, collecter des preuves, évaluer les contrôles et générer des rapports d’audit.

---

## ⚙️ Technologies utilisées

### Frontend

* React (Vite)
* JavaScript (ES6+)
* CSS

### Backend

* Node.js
* Express.js

### Base de données

* SQL

---

## 🎯 Objectif

Faciliter l’apprentissage et la simulation des audits ISO 27001/27005 dans un environnement interactif.
